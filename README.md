# The Beastly Bite - Restaurant Website

Welcome to the digital home of **The Beastly Bite**, where we unleash your wildest hunger!

## Features

- **Modern UI/UX**: Designed with a "beastly" theme using dark slate and vibrant red accents.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Dynamic Menu**: Interactive menu with categories and real-time "Add to Cart".
- **Shopping Cart**: Full cart management with quantity updates and order placement simulation.
- **Animations**: Smooth transitions powered by Framer Motion.

## Tech Stack

- **Framework**: React 18 with Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Language**: TypeScript

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/App.tsx`: The main application logic and layout.
- `src/data.ts`: Mock menu items and data.
- `src/types.ts`: TypeScript interfaces for menu and cart items.
- `src/index.css`: Global styles and Tailwind directives.
